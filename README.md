## 요구환경

- JDK 17
- Gradle 8.5

## 실행방법

### IDE에서 실행

1. git clone

```bash
$ git clone https://github.com/SWE-PRJ/console.git
```

2. IDE를 통해 gradle 프로젝트 빌드한 후 Main.java 실행

### 빌드 파일 실행

1. jar 파일 실행

```bash
$ cd build/libs
$ java -jar {jar파일명}.jar
```
